# Implementation

This folder is reserved for the buildable `003-client-server-spa-api` example implementation.

The implementation has not started yet.

Planned implementation baseline:

- React SPA frontend
- ASP.NET Core API backend
- PostgreSQL database
- same-origin cookie authentication
- single-application-first production direction
- project delivery and team collaboration domain
- automated test coverage as part of the MVP

Expected first implementation phases:

1. foundation and auth baseline
2. projects and tasks
3. comments, dashboards, and filtering
4. testing and release hardening

Suggested future structure:

```text
implementation/
  src/
    backend/
    frontend/
  tests/
    backend/
    frontend/
    smoke/
  infra/
  docker/
```
